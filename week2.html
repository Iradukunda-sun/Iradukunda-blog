WEEK 2

We dived into CSS with the introduction and a look at most of the features before getting into bootstrap and the bootstrap grid system. We also looked at GIT in the cloud.

It was a thrilling experience with some hurdles and uphill tasks that taught us the value of hardwork and resilience.

We moved into the habits that can help us achieve interdependence and great networks through public victory. These habits include; think win-win, seek first to
understand then to be understood and synergize. All these habits would only be possible if one remembered to sharpen the saw everyday.

It was such an interactive class with various discussions and alot of inspiring stories shared within the class.
